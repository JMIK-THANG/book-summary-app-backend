# Book Summary App Backend

## Overview

This is the backend for the Book Summary App built with Node.js, Express, and PostgreSQL.

## Features

- User authentication
- Google OAuth login
- CRUD operations for books
- Comments API
- PostgreSQL database

## Tech Stack

- Node.js
- Express.js
- PostgreSQL
- Google OAuth
- Multer
- Cloudinary (optional)

## Installation

```bash
npm install
npm start
```

## Environment Variables

Create a `.env` file and add:

```env
DATABASE_URL=your_database_url
GOOGLE_CLIENT_ID=your_google_client_id
GOOGLE_CLIENT_SECRET=your_google_client_secret
```
